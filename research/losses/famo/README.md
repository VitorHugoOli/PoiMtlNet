# famo

Why this?
- FAMO is a low-overhead adaptive weighting strategy that avoids per-step
  expensive multi-objective solvers.

Runtime mapping:
- Registry key: `famo`
- Runtime class: `losses.mtl_baselines.FAMOLoss`

Source:
- In-repo implementation: `src/losses/mtl_baselines.py`
- Paper: Fast Adaptive Multitask Optimization (NeurIPS 2023)
