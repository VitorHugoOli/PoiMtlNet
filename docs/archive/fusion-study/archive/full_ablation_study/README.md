# Full MTLnet Ablation Study: Fusion + Alabama

## Purpose

Comprehensive ablation study for the MTLnet multi-task learning framework,
testing all combinations of architectures, optimizers, and task heads on the
fusion embedding engine (128-dim, concatenation of HGI + Space2Vec).

This study provides the experimental backbone for the research article.
