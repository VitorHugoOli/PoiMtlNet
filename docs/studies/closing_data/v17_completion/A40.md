## Queue (fast → slow)

### A40-1 · STL cat ceiling re-tune (v17) — **fastest; offload from the H100**
Same job as **H2** in [`H100.md`](H100.md) (STL cat ceiling re-tuned to v17, 5 states × {0,1,7,100}). It is trivially
cheap (7-class `next_gru`, ~100 s/run) and does not need the big card, so run it here to free the H100 for the heavy
MTL/Istanbul work. Recipe + acceptance: see H2. **Do the small states (AL/AZ) first** (seconds), then FL, then CA/TX.
```bash
for S in 0 1 7 100; do for ST in alabama arizona florida california texas; do
  python scripts/train.py --task next --state $ST --engine check2hgi_dk_ovl \
      --model next_gru --folds 5 --epochs 50 --seed $S \
      --batch-size 8192 --onecycle-per-head-lr --cat-lr 1e-3 --max-lr 3e-3
  python scripts/closing_data/score_stl_cat_ceiling.py <rundir> --tag ${ST}_s${S}_stl_cat_ceiling_v17
done; done
```
