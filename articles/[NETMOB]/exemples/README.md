19 de junho de 2026.

Os quatro arquivos individuais foram extraídos, sem alteração de conteúdo, das páginas 27–28,
98–99, 125–126 e 196–197 do livro oficial. Eles são os abstracts NetMob de duas páginas, não versões
full paper.

Os PDFs foram normalizados para **PDF 1.4**, com nova tabela de referências e remoção de marcações e
links internos inválidos presentes no arquivo original. Isso melhora a compatibilidade com Preview,
Acrobat, navegadores e visualizadores integrados.

## Integridade

Todos os PDFs locais devem passar por `pdfinfo` e extração de texto com `pdftotext`. O livro completo
é a fonte canônica; os arquivos individuais existem apenas para facilitar leitura e comparação.
