## Limitações da coleta

Os proceedings ACM de 2020–2023 são volumes comerciais. Não existe um “livro completo”
publicamente baixável identificado para todas as edições. Os diretórios `exemples/`
armazenam catálogos e apontadores oficiais; PDFs integrais só devem ser adicionados quando
o autor/editora disponibilizar uma cópia pública legítima. O ambiente de execução também
bloqueou a transferência binária direta, por isso os links públicos foram preservados em
vez de criar arquivos incompletos ou contornar direitos de acesso.
