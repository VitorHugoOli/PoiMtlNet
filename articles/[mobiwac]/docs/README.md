## Escopo e cautelas

A janela principal é 2020–2025: 2020, 2021, 2022, 2023 e 2025. Não houve edição em
2024. A edição de 2025 foi publicada dentro dos proceedings IEEE de MSWiM 2025, e a
MobiWac 2026 usa template IEEE para papers regulares.

A organização não publica uma rubrica completa de revisão. Este dossiê separa
explicitamente regras oficiais de inferências feitas a partir dos papers e programas.
