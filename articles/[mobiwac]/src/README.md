# MobiWac 2026 paper source

`main.tex` is the root document (IEEE two-column conference, `IEEEtran` class).
It assembles the paper by `\input`-ing the content sources: prose lives in
`sections/*.tex` (8 files, `01_introduction` … `08_conclusion`), tables in
`tables/*.tex` (3), and figures in `figs/*.tex` (4). Bibliography keys live in
`references.bib`. The TikZ figures (`figs/fig1_dataflow.tex`, `figs/fig2_model.tex`)
are `\input` inline; the data figures (`figs/fig3_embquality.tex`,
`figs/fig4_deltas.tex`) `\includegraphics` the PDFs `figs/fig3_embquality.pdf`
and `figs/fig4_deltas.pdf`, which are **required build inputs** generated by the
companion scripts `figs/fig3_embquality.py` and `figs/fig4_deltas.py`.

`IEEEtran.cls` is bundled here so the paper compiles locally without a full
TeX Live IEEE install.

## Compile

```sh
pdflatex -interaction=nonstopmode main.tex
bibtex main
pdflatex -interaction=nonstopmode main.tex
pdflatex -interaction=nonstopmode main.tex
```

This produces `main.pdf` (9 pages). Build artifacts (`*.aux`, `*.log`, `*.bbl`,
etc.) are gitignored; the figure PDFs and `main.pdf` are kept under version
control.
