# random_weight

Why this?
- Stochastic weighting is a cheap control baseline to verify whether complex
  adaptive optimizers are truly helping.

Runtime mapping:
- Registry key: `random_weight` (alias: `rlw`)
- Runtime class: `losses.mtl_baselines.RandomWeightLoss`
